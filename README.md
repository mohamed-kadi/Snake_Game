# Snake Game

A browser-based snake game built with React + Vite.

This project is being developed as a learning journey with a long-term goal:
publish a polished version to the Google Play Store.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+ (LTS recommended)
- npm

### Run locally

1. Install dependencies:
   `npm install`
2. Start development server:
   `npm run dev`
3. Open the local URL shown in the terminal (usually `http://localhost:3000`)

## Available Scripts

- `npm run dev` - Start local development server
- `npm run build` - Build production bundle
- `npm run preview` - Preview production build locally
- `npm run lint` - Run TypeScript checks

## Learning Roadmap (toward Google Play)

- Build a solid game core (controls, game loop, scoring, game states)
- Improve UX (sound, themes, accessibility, mobile responsiveness)
- Add persistence (high scores, settings)
- Convert to mobile app packaging flow (for Android)
- Prepare assets and release checklist for Play Console
